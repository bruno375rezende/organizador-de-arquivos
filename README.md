Organizador de Arquivos 🗂️

Script Python que automatiza a organização de arquivos por extensão (.txt, .xlsx, .docx, .png, .jpg, .pdf).

Move arquivos automaticamente para pastas específicas.

Gera log com data, hora e destino de cada movimentação.

Ideal para manter seus documentos e imagens organizados rapidamente.

Como usar:

Coloque os arquivos na pasta descompact.

Execute o script:

python file.organizer.py


Confira os arquivos organizados nas pastas e o log em registro.log.

Funcionalidades extras:

Criação automática das pastas de destino se não existirem.

Mantém os nomes originais dos arquivos.

Log detalhado de todas as operações.
